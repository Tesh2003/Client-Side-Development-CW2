🌊 Marine Guardians: Protecting Marine Life
📌 Project Overview

This project is a multi-page responsive website focused on raising awareness about ocean pollution, marine conservation, and endangered marine species. The website was developed as part of a client-side development coursework and demonstrates the use of HTML, CSS, and JavaScript to create an interactive and visually engaging user experience.

The primary goal of the website is to educate users about environmental issues affecting the ocean while also encouraging participation in conservation efforts. The design combines informative content with modern UI features such as animations, video backgrounds, and interactive components to improve user engagement.

🎯 Objectives
Raise awareness about ocean pollution and its effects
Highlight endangered marine species and ecosystems
Promote sustainable fishing and conservation practices
Provide an interactive and visually appealing user experience
Apply front-end development skills using HTML, CSS, and JavaScript
🧩 Website Structure

The website consists of multiple interconnected pages:

🏠 Home Page

The homepage acts as the central hub of the website. It features a full-screen video background to immediately capture user attention and create a strong visual impact. Navigation links are clearly displayed, allowing users to access all other sections easily.

🌍 Ocean Pollution & Its Effects

This page provides detailed information about the causes and impacts of ocean pollution. It includes:

Causes such as oil spills, plastic waste, and industrial runoff
Environmental and health impacts
Supporting images for visual understanding

The layout is structured using sections and headings to improve readability and content organisation.

🐋 Risk of Extinction

This page highlights marine species that are currently endangered, including:

Blue Whale
Hammerhead Shark
Sea Turtles
Dugong
Vaquita

Each species is presented with a short description and image. The page also includes a “Back to Top” button, implemented using JavaScript, to improve navigation.

🪸 Why Coral Reefs Matter

This section explains the importance of coral reefs in marine ecosystems. It covers:

How coral reefs are formed
Their role in supporting biodiversity
Types of coral reefs (hard and soft corals)
Environmental threats such as climate change and pollution

The page uses structured content and image galleries to enhance user understanding.

🎣 Fishing and Marine Conservation

This page focuses on sustainable fishing and conservation strategies. It explains:

The importance of sustainable fishing practices
Overfishing and its consequences
Conservation methods such as Marine Protected Areas (MPAs)
Pollution control and environmental protection efforts

The page combines text and images to clearly communicate key ideas.

🤝 Volunteer Page

The volunteer page encourages users to participate in environmental activities. It includes:

Informational content about volunteering
A fading text animation effect to draw attention to key messages

This animation is implemented using CSS transitions, improving user engagement without overwhelming the interface.

📝 Feedback Page

The feedback page allows users to share their experience through an interactive form. Features include:

Input fields for name, email, and message
Yes/No toggle buttons with visual selection states
Star rating system using icons
Animated star click effect
JavaScript alert confirmation on submission

This page demonstrates user interaction handling using JavaScript.

📊 Accomplishments Page

This page presents a timeline of beach clean-up events. Features include:

Timeline layout for structured presentation
Image scrolling carousel effect
Repeated image tracks for smooth animation

It visually showcases the impact of conservation activities.

👥 Team Page

The team page introduces the members involved in the project and their roles. It helps personalise the website and provides credit to contributors.
